# Django-React


## Clone repository
```bash
$ mkdir demo
$ cd demo
$ git clone https://github.com/Rahulkardam9358/django-react.git
```

## Install dependencies
```bash
$ cd django-react
$ pipenv install
$ cd frontend
$ npm install

# run frontend
$ npm run dev
```

## Run project (in django-react dir)
```bash
$ ./manage.py migrate
$ ./manage.py runserver
```
